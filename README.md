groovyKata
==========

My coding exercises in Groovy.

Pre-requisties
==============
Gradle - http://www.gradle.org/

Compile
=======
`$ gradle compileGroovy`

Run Test
========
`$ gradle test`

IntelliJ IDEA
=============
If you are using IntelliJ IDEA, run
`$ gradle idea`

and execute the resulting generated *.ipr file.
